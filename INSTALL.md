# 安装和使用指南

## 快速开始

### 1. 准备图标（可选但推荐）

在 `icons/` 文件夹中创建三个PNG图标文件：
- `icon16.png` - 16x16像素
- `icon48.png` - 48x48像素
- `icon128.png` - 128x128像素

**快速创建图标的方法：**
- 使用在线图标生成器：https://www.favicon-generator.org/
- 或使用简单的文档图标
- 如果没有图标，插件仍可工作，但会显示默认图标

### 2. 安装到360浏览器

1. **打开扩展管理页面**
   - 方法1：点击浏览器右上角菜单（☰） → 更多工具 → 扩展程序
   - 方法2：直接在地址栏输入 `chrome://extensions/` 或 `se://extensions/`

2. **启用开发者模式**
   - 在扩展程序页面右上角，打开"开发者模式"开关

3. **加载插件**
   - 点击"加载已解压的扩展程序"按钮
   - 选择 `paper_organizer_extension` 文件夹
   - 点击"选择文件夹"

4. **确认安装**
   - 插件应该出现在扩展程序列表中
   - 浏览器工具栏应该显示插件图标

### 3. 使用插件

#### 下载论文

1. 访问任何包含PDF链接的网页（如arXiv、Google Scholar等）
2. 点击PDF下载链接
3. 插件会自动：
   - 拦截下载请求
   - 解析PDF元数据
   - 提取作者、日期等信息
   - 自动重命名文件并下载

#### 查看已下载论文

1. 点击浏览器工具栏上的插件图标
2. 打开管理面板
3. 查看所有已下载论文的列表
4. 点击论文链接可查看原始网页

#### 调试信息

打开浏览器开发者工具（F12），在控制台中可以看到：
- `[论文助手] 检测到PDF下载链接: [URL]`
- `[论文助手] 开始处理PDF: [URL]`
- `[论文助手] PDF元数据提取成功: [数据]`
- `[论文助手] 准备下载并重命名为: [文件名]`

## 常见问题

### Q: 插件无法加载？
**A:** 检查以下几点：
- 确保开启了"开发者模式"
- 检查 `manifest.json` 文件是否存在且格式正确
- 查看扩展程序页面的错误信息（红色错误提示）

### Q: PDF无法解析？
**A:** 可能的原因：
- 网络连接问题（需要从CDN加载pdf.js）
- PDF链接有CORS限制（会回退到基础下载模式）
- 查看控制台错误信息

### Q: 文件没有重命名？
**A:** 可能的原因：
- PDF不包含可提取的元数据
- 日期/作者信息格式不标准
- 查看控制台日志了解提取结果

### Q: 如何清除所有记录？
**A:** 在管理面板中点击"清空记录"按钮

## 测试建议

### 测试网站推荐

1. **arXiv** (https://arxiv.org/)
   - 包含大量PDF论文链接
   - 元数据通常完整

2. **Google Scholar** (https://scholar.google.com/)
   - 学术论文搜索
   - 点击PDF链接测试

3. **IEEE Xplore** (https://ieeexplore.ieee.org/)
   - 需要登录，但PDF格式标准

### 测试步骤

1. 访问测试网站
2. 找到一个PDF论文链接
3. 点击链接
4. 打开开发者工具（F12）查看控制台输出
5. 检查下载文件夹中的文件是否已重命名
6. 点击插件图标查看管理面板

## 卸载插件

1. 打开扩展程序页面
2. 找到"论文智能整理助手"
3. 点击"移除"按钮
4. 确认删除

**注意：** 卸载插件不会删除已下载的PDF文件，但会清除存储的论文记录。
