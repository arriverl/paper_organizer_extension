# 图标文件说明

## 已配置的图标文件

插件已配置使用以下图标文件：

- ✅ `icon16.png` - 16x16 像素（从 favicon-16x16.png 复制）
- ✅ `icon48.png` - 48x48 像素（从 android-icon-48x48.png 复制）
- ✅ `icon128.png` - 128x128 像素（从 android-icon-192x192.png 复制）

这些图标文件已从现有的图标资源中创建，插件可以直接使用。

## 图标来源

图标文件基于现有的 favicon 和 Android 图标资源：
- `icon16.png` ← `favicon-16x16.png`
- `icon48.png` ← `android-icon-48x48.png`
- `icon128.png` ← `android-icon-192x192.png`

## 自定义图标

如果需要更换图标，可以：
1. 替换 `icon16.png`、`icon48.png`、`icon128.png` 文件
2. 确保新图标尺寸正确（16x16、48x48、128x128）
3. 重新加载插件以应用新图标

## 其他可用图标

此文件夹还包含其他尺寸的图标文件，可用于其他用途：
- Apple 图标系列（57x57 到 180x180）
- Android 图标系列（36x36 到 192x192）
- Microsoft 图标系列（70x70 到 310x310）
- Favicon 系列（16x16 到 96x96）
